--- layout: post title: 'Start eclipse from the command line ' date: '2010-09-09T10:28:00+02:00' tags: - eclipse tumblr_url: http://andywenk.tumblr.com/post/1090770845/start-eclipse-from-commandline --- Hi, just a short tip if you want to start eclipse from the command line. You can find the possible options here: http://help.eclipse.org/ganymede/index.jsp?topic=/org.eclipse.platform.doc.user/tasks/running_eclipse.htm Example: 
#!/bin/bash
 
$HOME/applications/ver/eclipse/eclipse -clean -nosplash \
-data $HOME/project/ver/
